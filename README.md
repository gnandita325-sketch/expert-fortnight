# Student Expense Analyzer

A Python-based command-line application designed to help students track and analyze
their daily expenses.

## Features
- Add daily expenses with category and notes
- Store expenses in a file
- View total and category-wise spending
- Simple and beginner-friendly design

## Concepts Used
- File Handling
- Functions
- Loops and Conditionals
- Dictionary for data analysis
- datetime module

## Language
- Python
